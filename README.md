# Grumpy Cat Resume

This project is a simple resume-style webpage created using HTML and CSS.

The goal was to practice structuring a webpage, organizing content, and applying basic styling. I focused on keeping the layout clean and easy to read while following the design we were given.

This project helped me get more comfortable with:
- HTML structure
- CSS styling
- Organizing files and folders
- Building a basic webpage from start to finish

If I had more time, I would improve the styling and make the layout more responsive for different screen sizes.
